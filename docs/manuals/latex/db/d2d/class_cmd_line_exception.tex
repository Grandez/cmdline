\doxysubsection{Descripción detallada}
Clase base para cualquier excepcion generada en el sistema. 

Distinguimos tres tipos de excepciones

A) Las que \char`\"{}solo\char`\"{} se pueden producir durante el tiempo de desarrollo B) Las que se producen en tiempo de ejecucion C) El usuario ha solicitado la ayuda del programa

Las primeras son derivadas de errores en el proceso de desarollo, por ejemplo, un error en el nombre del parametro. Estas excepciones derivan de runtime\+\_\+error Las segundas son derivadas del proceso de analisis en tiempo de ejecución, dado que C++ ya define una invalid\+\_\+argument excepcion {\ttfamily \mbox{\hyperlink{class_cmd_line_exception}{Cmd\+Line\+Exception}}} deriva de esta clase

A su vez, exisena clases especializadas en funcion del tipo de error que se haya producido
